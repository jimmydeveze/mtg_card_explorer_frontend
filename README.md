# MTG Card Explorer

Explorador de cartas de Magic: The Gathering desarrollado con React y conectado a una API third-party.

## Tecnologias:

- React: El framework que nos ayudara para el desarrollo de este proyecto, que nos ayudara a crear componentes para una mejor modularidad del proyecto
- Vite: herramienta de construcción (build tool) de próxima generación para el desarrollo frontend
- Javascript: es parte fundamenta de React, ya que que aquise combiana el HTML, con Java =script para crear los modulos, esots archivos tien la extencion de jsx
- CSS: para dar estilos a los elementos y pagian
