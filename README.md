# MTG Card Explorer

Explorador de cartas de Magic: The Gathering desarrollado con React y conectado a una API externa para obtener información en tiempo real sobre cartas oficiales del juego.

La aplicación permite buscar cartas por nombre, aplicar filtros avanzados y consultar detalles completos de cada carta, incluyendo imagen, tipo, rareza, coste de maná y más.

## Tecnologias:

- React: El framework que nos ayudara para el desarrollo de este proyecto, que nos ayudara a crear componentes para una mejor modularidad del proyecto
- Vite: herramienta de construcción (build tool) de próxima generación para el desarrollo frontend
- Javascript: es parte fundamenta de React, ya que que aquise combiana el HTML, con Java =script para crear los modulos, esots archivos tien la extencion de jsx
- CSS: para dar estilos a los elementos y pagian

## API

Uso de la API SScryfall: El proyecto utiliza la API pública de Scryfall, una de las bases de datos más completas de cartas de Magic: The Gathering.

- Scryfall proporciona:
  - Información oficial de cartas
  - Imágenes en alta calidad
  - Coste de maná - Tipo y subtipo
  - Rareza
  - Legalidad en formatos
  - Datos actualizados en tiempo real

- La aplicación consume esta API para:
  - Obtener cartas aleatorias
  - Realizar búsquedas por nombre
  - Aplicar filtros personalizados
  - Implementar paginación infinita

- Documentación oficial:
  https://scryfall.com/docs/api
